## 统一配置说明

颜色均使用以下颜色，后续可继续前往http://lcpmgh.com/colors/ 进行配色选取

'zstd': '#5CA7C7', 'brotli': '#FBCE6A', 'gzip': '#383838','Co4U': '#D4352D'



为确保图片的大小易于调整，我们暂时规定所有图片的宽度为8，高度为6，即为

plt.figure(figsize=(8/2.54, 6/2.54))

字体除了图例均需保证为10，图例可以根据大小调整，选择loc='best'

保存时的dpi = 600
